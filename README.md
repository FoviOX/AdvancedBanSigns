## AdvancedBanSigns

# Features
# Small plugin size
# Optimized
# Cancel place sign, if player muted

#Dependencies:
 Advancedban

#How to install:
1) Put plugin AvancedBanSigns in plugin folder
2) Restart/Reload server
3) Done!

#Config:
Message-On-Place-Sign: |-
        &7|
        &7| &cYou are muted by &r%player%.
        &7| &cReason &r%reason%
        &7| &cTime &r%time%
        &7|
